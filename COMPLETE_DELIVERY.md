## 🚀 Integration (3 Simple Steps)

### Step 1: Add CSS Link

```html
<!-- In your <head> tag, after styles.css -->
<link rel="stylesheet" href="timeline-styles.css" />
```

### Step 2: Add HTML Section

```html
<!-- In your <main> where #timeline anchor is -->
<section id="timeline" class="timeline section">
  <!-- Copy entire content from timeline-section.html -->
</section>
```

### Step 3: Add JavaScript

```html
<!-- Before closing </body> tag -->
<script src="timeline-script.js"></script>
```

## 🎨 Customization Guide

### Change Colors

```css

--secondary: #0099ff; /* Blue */
```

### Adjust Spacing

```css

  margin-bottom: 80px; /* Gap between items */
}
```

### Modify Animation Speed

```css

                                 ↑ seconds
```
